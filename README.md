---

## 📌 Project Overview

This Weather Website allows users to:

- 🔍 Search for any city
- 🌡 View current weather conditions
- 📅 Display 7-day forecast
- 🌥 Show weather status and icons
- 🎨 Automatic background image change
- 📱 Fully responsive design

The application fetches live weather data from WeatherAPI.

## 🚀 How It Works

1. User enters a city name.
2. JavaScript sends a request to WeatherAPI.
3. The API returns weather data in JSON format.
4. Data is dynamically rendered on the page.
5. Background images change automatically every 2 seconds.
